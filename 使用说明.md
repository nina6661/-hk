# 香港出入境数据统计系统

## 🌐 访问方法

### 方法1：本地访问（推荐）

**步骤1：打开终端**

在Finder中找到 `应用程序 → 实用工具 → 终端`，或使用Spotlight搜索"终端"

**步骤2：运行启动命令**

复制并粘贴以下命令到终端中：

```bash
cd /Users/didi/dchat-workplace/hk-border-stats
python3 run.py
```

**步骤3：打开浏览器**

在浏览器中访问：**http://localhost:5000**

---

### 方法2：公网访问（分享给他人）

如果你想让其他人也能访问，需要使用内网穿透工具。

**选项A：使用ngrok（推荐）**

1. 打开一个新终端窗口
2. 运行以下命令：

```bash
# 先启动Flask（在第一个终端窗口）
cd /Users/didi/dchat-workplace/hk-border-stats
python3 run.py

# 然后在另一个终端窗口启动ngrok
ngrok http 5000
```

3. ngrok会显示一个公网URL，类似：`https://xxxx-xxx-xxx.ngrok-free.dev`
4. 将这个URL分享给其他人即可访问

**选项B：部署到云服务器（长期方案）**

如果你需要长期公网访问，建议部署到云服务器：

- **PythonAnywhere** (免费)：https://www.pythonanywhere.com
- **Render** (免费)：https://render.com
- **Railway** (免费额度)：https://railway.app

## 🔄 数据更新

系统会在每周一自动：
- 09:00 同步最新数据
- 12:00 生成周报

也可以手动更新：
```bash
curl -X POST http://localhost:5000/api/sync-data
```

## 💡 常见问题

### Q: 如何停止服务器？
A: 在运行服务器的终端中按 `Ctrl+C`

### Q: 端口被占用怎么办？
A: 修改 `config.py` 中的 `PORT = 5000` 为其他端口

### Q: 数据如何备份？
A: 复制 `database/border_stats.db` 文件即可
