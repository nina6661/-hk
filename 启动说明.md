# 香港出入境数据统计系统 - 启动说明

## ✅ 系统已初始化完成

- **数据记录**: 58,468 条
- **数据范围**: 2021-01-01 至 2026-05-20
- **数据库**: SQLite (database/border_stats.db)

## 🚀 启动方法

### 方法1: 使用启动脚本（推荐）

打开终端，执行：

```bash
cd /Users/didi/dchat-workplace/hk-border-stats
./start_server.sh
```

### 方法2: 直接运行Python

```bash
cd /Users/didi/dchat-workplace/hk-border-stats
python3 run.py
```

### 方法3: 使用Flask命令

```bash
cd /Users/didi/dchat-workplace/hk-border-stats
export FLASK_APP=app.py
python3 -m flask run --host=0.0.0.0 --port=5000
```

## 🌐 访问地址

启动后，在浏览器中访问：

- **首页（最新周报）**: http://localhost:5000/
- **历史数据**: http://localhost:5000/history
- **数据预测**: http://localhost:5000/prediction

## 📊 功能说明

### 首页 - 最新周报
- 自动展示上周（周一至周日）完整数据报告
- 出入境流量统计表格
- 年度对比折线图（入境/出境各一张）
- 周度汇报小抄（可直接朗读）

### 历史数据页面
- 按周查询：选择日期自动匹配自然周
- 自定义区间：选择起止日期生成专属报表

### 数据预测页面
- 基于30天历史数据预测未来客流
- 支持7/14/30天预测

## 🛑 停止服务器

在运行服务器的终端中按 `Ctrl+C`

## 🔄 数据更新

系统会在每周一自动：
- 09:00 同步最新数据
- 12:00 生成周报

也可以手动触发：
```bash
curl -X POST http://localhost:5000/api/sync-data
```

## ❓ 常见问题

### Q: 端口被占用怎么办？
A: 修改 `config.py` 中的 `PORT = 5000` 为其他端口

### Q: 数据多久更新一次？
A: 每周一自动更新，也可以手动触发

### Q: 如何查看日志？
A: 服务器运行时会在终端显示日志

## 📧 技术支持

如有问题，请检查：
1. Python版本 >= 3.8
2. 依赖已安装（requirements.txt）
3. 数据库文件存在（database/border_stats.db）
